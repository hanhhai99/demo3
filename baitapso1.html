5 NGÔN NGỮ LẬP TRÌNH WEB PHỔ BIẾN NHẤT HIỆN NAY

1.PHP

Ưu điểm:
  • Dùng mã nguồn mở và ổn định
  • Phổ biến
  • Dễ học khi đã biết HTML, CSS
  • Dựa vào XAMP
  • ĐI cặp với
  • Linh hoạt trong khả năng phát triển web
Nhược điểm
  • Mã nguồn không đẹp
  • chỉ chạy trên ứng dụng web

2.Python

Ưu điểm:
  • có hình thức sáng sủa
  • Có trên tât cả nền tảng của hệ điều hành
  • Tốc đổ xử lý nhanh
  • Tương thích mạnh mẽ với UNIX với số lượng tư viện khổng lồ
Nhược điểm
  • Không cho các thuộc tính: protect, private, public
  • Không có vòng lặp do while hay switch case
  • tốc độ xử lý nhanh hơn php nhưng chậm hơn java

3.Java

ưu điểm
  • Mã nguồn rõ ràng, tách biệt với HTML
  • Visual Studio có thể sinh mã, tiết kiện thời gian viết code
  • Dễ học khi biết HTML. Có thể dùng PHP, Ruby để GUI
  • Tương thích trên mọi nền tảng, an toàn, mạnh mẽ. Cú pháp rõ ràng mạch lạc
Nhược điểm:
  • Tốc độ hơi bị chậm nhưng chấp nhận được
  • config nhiều

4. JavaScript

Ưu điểm:
  • Thiết kế độc lập với HĐH
  • Dễ dàng tương tác, điều khiển và tránh bớt việc xử lý từ phía server
  • nắm vững kiến thức JavaScript rất hữu dụng cho bạn sau này có thể tiếp thu những công nghệ mà...
Nhược điểm:
  • JS không có trình biên dịch riêng mà nó được diễn dịch và chạy bở trình duyệt hỗ trợ nó
  • Có thể làm ứng dụng Web của bạn trở nên nặng nề hơn
  • Bảo mật kém, không có khả năng giấu mã

5. C++

Ưu điểm:
  • Sử dụng được ở mọi nơi và kế thừa dduojc các điểm mạnh của C
  • Là ngôn ngữ lập trình hướng đối tượng rất mạnh
  • C++ là ngôn ngữ có ít từ khóa, tạo thuận lợi cho việc học
  • Với cấu trúc module cho phép sử dụng nhiều lần các chương trình con dưới dạng hàm
  • Có nhiều thư viện sẵn có cho việc thêm các chức năng
Nhược điểm:
  • Khá khó học
  • Chương trình chạy chậm hơn chương trình C
  • Tương tác ngược với C nên làm hạn chế khả năng của nó
